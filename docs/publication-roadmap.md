# Publication Roadmap

This repository is being prepared as a manuscript-linked toolkit. Before public release, the following items should be completed:

- Add final paper title, author list, DOI, and citation metadata.
- Replace placeholder method text with manuscript-aligned wording.
- Publish a stable taxonomy export or release artifact if approved.
- Add screenshots or figures from the public demo workflow.
- Finalize license and archival release.
- Confirm GitHub Pages content matches the accepted manuscript.
